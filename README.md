## Run Locally

Clone the project

```bash
  git clone https://github.com/juniorimam/bookshelf-apps-react
```

Go to the project directory

```bash
  cd bookshelf-apps-react
```

Install dependencies

```bash
  npm install or npm i
```

Start the json-server

```bash
  npm run server
```

Start the Bookshelf Apps

```bash
  npm run start
```
